## Vereisten

Ik verwacht dat volgende software reeds geïnstalleerd is:

- [NodeJS](https://nodejs.org)

## Opstarten

- maak een .env file met deze inhoud:

    VITE_API_URL=`http://localhost:9000/api`

- yarn install

- yarn dev
